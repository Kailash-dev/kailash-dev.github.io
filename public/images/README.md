# Headshot photo

Drop your professional headshot here as:

```
kailash-headshot.jpg
```

## What we need

| Requirement | Detail |
|-------------|--------|
| **Filename** | `kailash-headshot.jpg` (or `.webp`) |
| **Location** | `public/images/kailash-headshot.jpg` |
| **Dimensions** | At least **800×800px**; ideal **1200×1200px** or **1200×1500px** (4:5 portrait) |
| **Format** | JPG or WebP (WebP preferred for performance) |
| **File size** | Under 500 KB after compression |
| **Crop** | Head and shoulders, centered face |
| **Background** | Neutral: white, light grey, or soft blur — matches minimal brand |
| **Lighting** | Even, natural light; avoid harsh shadows |
| **Expression** | Calm, approachable, confident — not overly corporate or casual |
| **Wardrobe** | Solid colours; avoid busy patterns |
| **Editing** | Light retouching only — no heavy filters |

## Style reference

Think **Stripe / Linear founder portraits**: clean, professional, human — not stock-photo stiff.

## After adding the file

The site automatically uses it in:
- Home hero (desktop)
- About page
- Falls back to **KG** initials until the file exists

No code changes needed — just add the file and refresh.
